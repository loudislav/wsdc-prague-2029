# Prague 2029 WSDC Bid

Minimal static landing page for Prague's bid to host the World Schools Debating Championship 2029.

## Files

- `index.html` — page content and metadata
- `styles.css` — design and responsive layout
- `assets/prague-charles-bridge-panorama.jpg` — CC0 background photograph by Fuchsbua

## Deploy

The website is fully static and does not require a build step.

- **Traditional hosting:** upload the contents of this directory to the web root.
- **GitHub Pages:** publish the repository from the root of the `main` branch.
- **Netlify / Cloudflare Pages:** connect the repository, leave the build command empty, and use `/` as the publish directory.

To preview locally, open `index.html` in a browser or run any simple static web server in this directory.
